Illustrating the creative process and approach.

RATIONALE

The second project in CCT360 provides students to design and develop a website using a well-established framework known as, Bootstrap. This website essentially represents my online resume in a creative and professional manner. With the emphasis on usability and accessbility, the website effectively applies both requirements in each page. As the bootstrap framework already provides the developer with established codes and various programming languages. For this particular project, HTML, CSS, and Javascript were heavily used.

The theme of the website is brown, earth tones to display a subtle color palette. The color brown is associated with an organic perspective and is illustrated as being reliable and approachable. With heavy emphasis on the brown shade, it is crucial to use only a few tints in each page.

NODE-LINK


The tree diagram illustrated above displays a visual representation of the professional website. The global and local navigation items have been precisely chosen to be simple but informative. These navigation items offer a broad range of genres that one can be interested in learning more about. In order to explore the navigation bar tools more effectively, I chose to have only one dropdown menu which consists of two inner pages.

CHOICE OF SELECTIONS

The homepage displays a working navigation to all five inner pages, as well as a footer. Using the readings provided for particularly this course, one of them outlines the principles of disclosure. This reading states that the landing page should have highlights that are essentially visual features that summarize the content presented in each inner page. The page consists of embedded, looped videos, images and written content. The inner pages consist of the content respective to its theme. All of the inner pages are different to one another, with a consistent theme and color palatte. From the listed page-typologies, four of tem were selected as well as the process page which is counted as the content-heavy page-type.

The following is a list of page-typologies used for this project:
'Home' - Landing page
'My Display' - Photo-rich page
'Videos' - Rich-media page
'Events' - Time-based page
'Contact' - Form-page
'Process' - Content-heavy page

DRAWBACKS

Some of the aspects of the pages worked out well however, there were some that were not working too well. The header in each inner page does not seem to be responsive when shrinked into a smaller screen. The second issue is the quality of the image and video files as the resolution had to be cut down.
